# C/C++ Codespace Template

เทมเพลตนี้ออกแบบมาเพื่อใช้งานร่วมกับ GitHub Codespaces

## เริ่มต้นใช้งานอย่างรวดเร็ว (Quick Start)

1.  **รันการตั้งค่า (Run Setup)**: ให้เปิด terminal แล้วพิมพ์คำสั่ง:
    ```bash
    bash .devcontainer/setup.sh
    ```
2.  **ขั้นตอนการเข้าสู่ระบบ (GitHub Login)**: เมื่อรันสคริปต์ข้างต้น ให้ทำตามขั้นตอนใน Terminal ดังนี้:
    - เลือก **Yes** เมื่อระบบถามว่า "Generate a new SSH key?"
    - กด **Enter** ข้ามขั้นตอน "Passphrase" (ไม่ต้องใส่รหัสผ่าน)
    - กด **Enter** ยอมรับชื่อ SSH key "GitHub CLI"
    - **คัดลอก (Copy) รหัส 8 หลัก** ที่ปรากฏบนหน้าจอ
    - กด **Enter** เพื่อเปิดหน้าเว็บล็อกอิน ให้วางรหัสที่คัดลอกมาแล้วกด Continue
    - เมื่อเสร็จแล้วระบบจะเชื่อมต่อกับ GitHub repository ส่วนตัวของเรา (`26cpprepo`)

## คุณสมบัติเด่น (Features)

-   **บันทึกอัตโนมัติ (Auto-Save)**: ระบบจะ commit และ push งานของเราไปยัง GitHub โดยอัตโนมัติหากไม่มีการแก้ไขไฟล์เกิน 30 นาที (Idle)
-   **การซิงค์ที่ปลอดภัย (Safe Sync)**: สคริปต์จะรวมงาน (merge) กับงานเดิมที่เรามีบน GitHub ให้อัตโนมัติ (ไม่มีการ force push ทับงานเก่า)
-   **พร้อมใช้งานทันที**: ตั้งค่าคอมไพเลอร์ `g++` และปุ่ม Run/Debug มาให้พร้อมใช้งาน

## การจัดการโฟลเดอร์

-   **`homework/`**: โค้ดและงานทั้งหมดของเราควรเก็บไว้ในโฟลเดอร์นี้
